- 확장 가능한 구조

### TREE.md 필수 항목
```markdown
# 프로젝트 구조

## 디렉토리 구조
project/
├── src/
│   ├── app/           # 애플리케이션 코어
│   │   ├── __init__.py
│   │   ├── main.py
│   │   └── ...
│   ├── core/          # 핵심 로직
│   ├── models/        # 데이터 모델
│   ├── services/      # 비즈니스 로직
│   ├── handlers/      # 요청 처리
│   └── utils/         # 유틸리티
├── tests/
│   ├── unit/
│   └── integration/
├── docs/
├── pyproject.toml     # 또는 package.json
└── README.md

## 파일 설명
| 경로 | 설명 |
|-----|------|
| src/app/main.py | 진입점 |
| src/core/config.py | 설정 관리 |
| ... | ... |
```
